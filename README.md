# Smart Healthcare – Hackathon Project

## Overview

This project was developed during a college hackathon where teams were assigned a domain and asked to build a working web application.

Our team was assigned the **Healthcare domain**, and we built a Smart Healthcare platform where users can search for doctors, view profiles, and book appointments.

## Features

* Doctor search system
* Doctor profile page
* Appointment booking interface
* Health dashboard UI
* Responsive animated design

## Technologies Used

* HTML
* CSS
* JavaScript
